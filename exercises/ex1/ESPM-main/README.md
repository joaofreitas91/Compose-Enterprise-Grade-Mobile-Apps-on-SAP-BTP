# Serviço CAP para ESPM

Este é um serviço básico do CAP para o modelo ESPM com dados de amostra.

## Implantação

Certifique-se de que você tenha uma instância de banco de dados SAP HANA Cloud em execução no espaço de sua subconta BTP para o qual deseja implantar (ou uma que seja compartilhada com o espaço)

Com MTA em um terminal:
```sh deployCFMTA.sh```

Alternativamente, usando CF Push:
```sh deployCFPush.sh```
